# Plugin Modernizer Stats

The [Plugin Modernizer Tool](https://github.com/jenkins-infra/metadata-plugin-modernizer) automates modernization of Jenkins plugin using OpenRewrite recipes. A [metadata-plugin-modernizer](https://github.com/jenkins-infra/metadata-plugin-modernizer) was established that stores raw modernization data in json/csv format generated by modernizer tool.
This static visualization site consumes this metadata and presents dashboard, per-plugin & per-recipe reports.
---

## Getting Started

> **Prerequisites** -- [Node.js](https://nodejs.org/) >= 22 (LTS) and npm >= 10.

```bash
# 1. Clone your fork of repository
git clone https://github.com/<your-user-name>/plugin-modernizer-stats.git
cd plugin-modernizer-stats

# 2. Install dependencies
npm ci

# 3. Start the development server
npm run dev
```

Open <http://localhost:5173/plugin-modernizer-stats/> in your browser.
