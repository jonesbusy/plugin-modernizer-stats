# Plugin Modernizer Stats

</div>

The [Plugin modernizer tool](https://github.com/jenkinsci/plugin-modernizer-tool) automates modernization of Jenkins plugins using OpenRewrite recipes. A [metadata-plugin-modernizer](https://github.com/jenkins-infra/metadata-plugin-modernizer) repository stores the raw modernization data in json/csv format generated by the tool. 
This static visualization site consumes this metadata and presents a dashboard with per-plugin and per-recipe reports.
In short, the overall work done by plugin modernizer till now in form of UI. 

## Getting Started

### Prerequisites  
- node.js (LTS version recommended)
- npm (latest stable version)
- git (for cloning repo & version control)

```bash
# 1. Clone your fork of the repository
git clone https://github.com/<your-user-name>/plugin-modernizer-stats.git
cd plugin-modernizer-stats

# 2. Install dependencies
npm ci

# 3. Start the development server
npm run dev
```
---

Open <http://localhost:5173/plugin-modernizer-stats/> in your browser.
